## 2024-05-24 - [Tailwind Configuration]
**Learning:** Tailwind v3 with shadcn/ui requires explicit color mapping in `tailwind.config.js` to match CSS variables in `index.css`.
**Action:** Ensure `tailwind.config.js` includes `theme.extend.colors` mapping when setting up shadcn/ui.

## 2024-05-24 - [Tailwind Version Compatibility]
**Learning:** react-scripts v5 is incompatible with Tailwind CSS v4.x.
**Action:** Always pin Tailwind CSS to v3.x (e.g., 3.4.17) in this environment.
